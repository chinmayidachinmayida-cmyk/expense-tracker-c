# Expense Tracker in C

## About
A simple Expense Tracker project developed using C programming and file handling.

## Features
- Add expenses
- View expenses
- Calculate total expenses
- Store data in file

## Technologies Used
- C Programming
- File Handling

## How to Run

gcc expense_tracker.c -o expense
.\expense.exe

## Future Improvements
- Expense categories
